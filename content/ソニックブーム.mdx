**ソニックブーム**（英：sonic boom）は、剣や弓矢で攻撃した座標に衝撃波で範囲攻撃をするエンチャントである。

## 入手方法
カスタムエンチャントの建物で
![カスタムエンチャントの建物](/files/Custom_Enchanting.png)

## 技術的情報
### ID
| 名称 | [リソース位置](https://ja.mc.wiki/w/リソース位置) | 翻訳キー |
| ---- | ---- | ---- |
| ソニックブーム | `custom_enchantments:exsp` | `enchantment.custom_enchantments.sonic_boom` |

## トリビア
- 帰還のエンチャントが実装されたとき、ソニックブームが付いた矢で自爆してしまう不具合があったので、今は一度目の着弾でのみ衝撃波が発生するように修正されている。
- 攻撃が直撃した場合、衝撃波のダメージが少し高くなるようになっている。
